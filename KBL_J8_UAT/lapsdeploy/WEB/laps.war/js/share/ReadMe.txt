For Alert Messages, 
	1)	include the AlertMessages.js file
	2)	call the function ShowAlert() method with errorcode as parameter.
		Syntax: ShowAlert(errcode) OR ShowAlert(errcode, value1 to replace $, value2...etc)

For Confirm Messages,

	1)	include the AlertMessages.js file
	2)	call the function ConfirmMsg() method with errorcode as parameter.
		Syntax: ConfirmMsg(errcode)
