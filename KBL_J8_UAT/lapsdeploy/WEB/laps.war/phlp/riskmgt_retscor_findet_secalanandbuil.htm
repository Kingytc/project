Securities as Land, Building etc Tab
Business rule ID - The loan product that was selected in “Step 3a” remains automatically selected. You can change it if required by just clicking the dropdown menu.
You can define upto five different ranges within which the value of security (land/ building) offered stands against the loan amount. Enter points for each range and description if needed. In our example, the housing loan:
Range from Range to Point Description
100.01 500 4 Value of land building to loan amount
75 100 2
0 74.99 0

Click “Save” to save changes
Click “ cancel” to cancel the operation. Data entered so far will not be saved in this case.
