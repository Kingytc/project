Setup

  • To start using LAPS you will have to create your organization in LAPS, different levels of your organization (hierarchical Groups) and users in each such group. They have to be defined by default. They can be defined through the setup
    This module is handled exclusively at the corporate office to set up primary business rules for operating the system. Only the user who has administration right can enter/ modify information in this module. Other users can access one or more sub menus in this module depending upon the access rights depending on the ‘Group’ he/ she belongs to.
