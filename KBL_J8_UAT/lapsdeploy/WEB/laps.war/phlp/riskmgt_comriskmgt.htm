If you are creating a new borrower ratio, then select “Borrower” from the “ratio applicable for” dropdown box
CMA type - Click the dropdown box, already created CMA formats are displayed here (these are already attached to various industries).
Select the CMA type for which you are creating the ratio.
Ratio description - Enter the ratio description in this alphanumeric field.
Ratio formula comparison - Click the dropdown field
Select between the two options available, namely, ‘Higher’, ‘Lower’
This is where the user defines, which way the ratios have to be in the comparison that happens in “Ratio formula for“
Ratio formula for - The ratios can be for “Historical” or “Projected” years. Historical is the last five years leaving out the current year. While projections is for the next 5 years excluding the current year. If “Historical” is chosen, the average of upto the last five years is taken and compared with that of last year. And in the same way if “Projections” is chosen then the average of the upto the next 5 years is compared with the next years ratio
