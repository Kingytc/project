Limit Master
This is the screen where the sanctioning powers of users are defined and later linked to access rights, workflow action definition. In Workflow, this User class is what authorizes a user to perform various actions including, approval/ rejection. Even if a person has the user class to approve a loan application, if the loan amount does not fall under the authority level of his user class then the system will not let him/ her take a decision on the loan application

    This screen is in "View" mode by default

    Select Existing Limits - You can create any number of loan amount ranges.
    To create a new loan range
    Click the dropdown box
    Click "New"
    Enter information in the following fields:
    Limit from - Enter the lower limit of the loan range.
    Limit to - Enter the upper limit of the loan range.
    User Class - Enter the class ID of the scale that is authorized to sanction the above mentioned loan limit.

    To Edit existing limits
    Click the dropdown box (select existing limits)
    Select the limit you wish to see
    Now click "Edit"
    Make necessary modifications
    Click "Apply" to save changes

    Click "Cancel" if you want to cancel the current operation
    Click "Close" to return to the "Corporate products setup" screen

    To Delete a record
    Select the required record from the dropdown box (select existing limit)
    Click "Delete" button that is found in the bottom of the screen
