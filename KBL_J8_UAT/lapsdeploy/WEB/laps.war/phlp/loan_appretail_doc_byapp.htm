Documents

Enter data in the fields provided
A set of loan product specific mandatory documents has to be submitted by the “applicant” and some are to be executed from the bank’s side. Applicant "displays both of these lists automatically under their respective sub-tabs, namely, and "By bank" in this tab called “Documents”. Information collected I s similar in both the sub-tabs.
Click “Edit” to enter any details
If you wish to submit a document then Right click on the square radio button next to the document. The system date gets populated automatically in the field “Received date” to be filled in by the user receipt. The document, which the user does not propose to obtain, the relevant radio button should not be clicked. However the user should give justification in brief in the relevant field for not selecting that document.
Renewal date - Enter the date on which the document needs to be renewed
Scan Status - The scan status of the document is automatically displayed in this field
Apart from the list of mandatory documents, which are loan product specific, you can also add new documents depending on the situation. In other words if you feel the current situation demands, then collect additional documents from the applicant. If a chosen document forms part of the loan product already defined then there is an alert that says, “Selected document is already available”. For this all you have to do is:
Click “Add”
A field called “Add document” appears along with a small “?” symbol next to it
Click the “?”, a list of document titles are displayed in a small pop-up screen
Select the title of the additional document you wish to attach
Click “Ok” in the pop-up screen
Click “Apply” to add the document to the already existing list of documents to be collected from either the applicant or he bank

Delete Documents - This function can be used to delete only the applicant specific additional documents created (if any).
Click “Apply” to save information entered so far or Click “Click” if there is nothing new to be saved. Now the delete button in the bottom of the page will be activated
Click the round radio button next to the “scan status” field next to the document you want to delete
Click “Delete” button displayed in the bottom of the page
Confirm the deletion
Remarks - This alphanumeric field is used for the credit officer to add comments incase of not selecting any of the documents as explained above
