    Editing an existing loan application

    Select “Existing” option from the Retail page.
    The following borrower search screen appears.

    Select the zone, region, branch from the list . Enter the borrower name and click the “Search” button.
    Click the respective borrower name.

    The following screen appears where you can search for a specific application number.
    Application Number : Enter the application number which you want to edit.
    Click Search button.
    To see the list of all the applications for the selected borrower, just click search without entering the application number.

    Application number of every form acts as a link, which can be clicked to view complete details of the application in another window with the following sub tabs:
    Applicant
    Co Applicant
    Guarantor
    Financial
    Coll. security
    Documents
    Edu. Loan
    Risk Analysis
    Appraisal
    Workflow
    Notes

    All the above sub tabs have been discussed in detail under the head "Create new application form"
    Click Reset button to clear the search result and to enable further search.
    On click of Close button confirmation is obtained and on acceptance of the same, navigation screen of LAPS is displayed.
