Inspection Report
On clicking “Inspection report” you will see a search page which allows you to select the applicant name followed by the application search screen. When the application number is clicked the following screen appears.

Select the year from the list box.
Click “New” to enter new details.
The following screen appears .

The general information of the borrower is populated from the application form.
Click “Edit”.
Enter the data for the following fields.
Purpose of Inspection
Date of inspection
Date of submission of report
Name of the officers concerned and their designation.

Click “apply” to save the data.
