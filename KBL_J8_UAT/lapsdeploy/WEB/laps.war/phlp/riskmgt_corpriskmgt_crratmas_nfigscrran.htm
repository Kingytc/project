Credit Rating Raster

Numerical Figure Score Range Tab

By default, Industry related parameters, ratios are configures, maintained and updated by the “Risk management” team. This ratios that are the end result of this module is compared against the range given in this “Numerical figure score range” screen and the corresponding point for the range is taken into the Rating model as the score for those parameters

Click the “?” next to the “Ratio text box
A small popup window comes up with a list of all ratios
Select the ratio for which you are defining the score range
Click the “?” next to the “Industry” text box
A small popup window displays the list of all the Industries defined so far
Select the Industry for which you are defining the score range
Range from - Range to - Enter the Score range
Point - Enter the points that each score range will earn
Click “Save” to save the changes made
Click “cancel” to cancel the operation
Click the next tab or click “Close” to close the screen and go back to the “commercial rating screen”

To edit an existing score range
Select the ratio (as explained above)
Select the Industry (as explained above)
Click “List”
The record is now displayed in the table where you had entered details
Click “Edit”
Make necessary changes
Click “Save” to save the changes made
Click “cancel” to cancel the operation
Click the next tab or click “Close” to close the screen and go back to the “commercial rating screen”
