Flowpoints
 This tab defines actions that are to be carried out while processing a loan application for this product.
•Duration within which a specific action is to be completed and the class of user who is eligible to complete the given action can be defined.
•The flow Point tab has the following three sub tabs, which defines the stage wise actions that are to be followed:
-In Process
-Post Approval
-Post Rejection
•Details regarding the following can be defined for each of the above three sub tabs:

    Flow point - List of serial numbers are displayed in this field.

    Action - Double click in the text box of 'Action' field, which is available in all the above three sub tabs. A pop up window with the list of various workflow actions is displayed (If you remember you would have defined all possible actions in the “flowpoints” under setup).
    Select the required action
    Enter the “Minimum class” of the user who can perform this action.
    Select the “Function” to which the user should belong to. The non-editable dropdown list, consists of the following options:
    Credit
    Financial
    Appraisal
    All
    Enter the time duration (days) within which the above mentioned “action” should be completed

    Note: LAPS is designed to keep track of the duration that an application is taking for every step. If this duration is exceeded, the particular action will automatically pass from the user defined for that action to the supervisor.
    In Process:
    •Mention all actions that are to be performed while the loan application is being processed
    Post Approval:
    •Mention all actions that are to be performed after the loan application has been approved
    Post Rejection:
    •Mention all actions that are to be performed after the loan application has been rejected
    You can click “Cancel” to cancel the change carried out
    Click “Close” to return to the setup screen
