Exposure
This screen allows to specify the maximum exposure limit for each group. The group master screen allows the user to search for the group or create a new group. To create a new group:

    Select New Group button
    Enter the details for the following fields.
    Type - Select the type from the list box
    Description - Enter the description for the group. For eg. Apollo.
    Maximum Exposure Limit in Rs. - Enter the maximum exposure limit
    Expiry Date - Select the expiry date from the calendar control.
    Click ‘apply’ to save.

    To edit a group :
    Click the search button.
    Select the group to be modified.
    Do the necessary changes.
    Click ‘Apply’ to save the changes.

    To delete a record
    Select the required record from the group master details screen.
    Click “delete” button that is found in the bottom of the screen
