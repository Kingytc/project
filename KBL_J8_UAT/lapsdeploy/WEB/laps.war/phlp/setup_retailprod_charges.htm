Charges
You can create multiple processing fee ranges based on the loan amount. In other words, for every loam amount range, you can define a new set of processing and documentation fee. With LAPS, you can define processing/ documentation charges by way of a range, percentage or a flat fee.

    By default the system control rests in “processing” sub-tab. Click on the “documentation” sub-tab once you are through with the processing sub-tab
    Click “Edit”
    Select processing/ documentation amount range - Click “new” to create a processing fee
    Amount range from - Enter the lower limit of the loan amount range
    Amount range to - Enter the lower limit of the loan amount range
    Processing fees (%) - You can define the processing/ documentation fee in %.
    Min processing/ documentation fees/ flat processing fee - Enter the minimum fee that may be collected for processing charges from the customer. This can also be a flat fee in which case a maximum amount need not be entered
    Max processing/ documentation fee - Enter the upper limit for the processing fee
    Charges can be defined in terms of % or as a range. These charges vary according to the loan product and the loan product is defined based on the amount range they fall into. e.g. For Document fees present slabs are Up to 25000 - Nil, >25000 to 2lacs - Rs150/- etc.
    If the documentation / processing fee is lower than the minimum fee mentioned then this amount is taken as the charge if the documentation / processing fee is higher than the maximum charge mentioned for that particular loan product then the “maximum processing/ documentation fee” is taken as the charge
    -Documentation/ Processing Charges are specified in %age terms the Min / Max need not be mandatory. The field “Select processing amount range” displays the amount range selected for various products.
    You can click “Cancel” to cancel changes carried out
    Click “Close” to return to the setup screen
