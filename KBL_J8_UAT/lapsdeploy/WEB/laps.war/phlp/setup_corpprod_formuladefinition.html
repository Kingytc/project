Formula Definition
This is the screen where various formulas that go into the financial analysis of commercial loan applications are created. The row descriptions that have been defined as "Calculated" in the "Row for" CMA master comes to life through the formulas that get defined in this "formula definition" screen
By default, the system will be in "New" mode in this screen. Which means you can start creating a formula directly without having to change modes.

    Creating a new formula

    CMA type - Choose the CMA for which the formula is to be defined

    Formula name - Enter the name of the formula. For example, Net sales

    Formula for - Click the dropdown box The entire list of financial modules gets listed (e.g. Financials, Ratios, Cash flow, funds flow, Assessment) Select the module for which you are going to create the formula

    Formula type - Click the dropdown box The options displayed are, Increase, Decrease, Group total, Net funds lost, Long term sources
    Choose the appropriate one Higher, Lower - When you are creating a formula by comparing the results of two other formulas in Cash flow then you can use this option. This sets a few preconditions, like:
         --If the result is in negative then the negative sign has to be removed and the value alone should be displayed.
         --If the result is in negative then the value should not be displayed

    Group total - This option is chosen when the sum of formulas which have used "increase" "decrease" option need to be found

    Year - The year you enter can be the financial analysis can be 'Current' or 'Previous'.

    Formula

    CMA 1 - Click the dropdown box
    All the parameters that have been defined in the CMA is listed
    Select the first parameter in the formula. For example, Sales
    Select operator - Click the dropdown operator
    All mathematical operators, namely, +, -, *,/ is displayed
    Select the operator you need for the formula. For example, you choose -

    CMA 2 - Click the drop down box
    All the parameters that have been defined in the CMA is listed here also
    Select the second parameter for the formula. For example, Excise duty
    Now click "Add formula"
    The formula you have created gets displayed in the textbox, "Formula". In our example, it is:
    Net sales = Sales - excise duty
    Click "Apply" to save the formula

    Editing a formula
    Select the CMA Type
    Select the "Formula name"
    The formula gets displayed in the text box, "Formula"
    Click "Edit"
    Make necessary changes
    Click "Apply" to save the changes

    Numerals - If any of the formulas have to be involved in a calculation with a numeral then this numeric field can be used to enter the relevant numeral. For instance if the "Net profit" has to be multiplied by 365, for some reason then 365 can be entered in "Numerals" field and the relevant formula can be chosen along with the relevant arithmetic expression

    Clear formula - Click this button to clear a formula you have created

    Formula creation using two formulas
    Click "New"
    Select the CMA type as mentioned above
    Enter the formula name in "Formula description"
    Formula 1 - Click the dropdown box
    Select the first formula in the calculation
    Select the mathematical operator
    Formula 2 - Click the dropdown box
    Select the second formula that is involved in creating a new formula
    Enter the Year
    Click "Add formula"
    The formula gets displayed in the text box, Formula
    Click "Apply" to save the new formula

    Click "Cancel" to cancel the operation. Remember that you will lose anything that you have entered
    Click "Close" to exit the screen and get back to the Corporate setup page
