Users
This option is for creating login name for employees/users of LAPS under various groups of the organization.
All employees who need access to LAPS have to be defined under this section. Login name, first name, last name, location, class, group and initial password should be entered for every user.


Note: The group chosen should contain the access and data security rights that you want to assign to the particular employee. Fields marked with ‘red’ star in the ‘Setup - Organization’ screen is mandatory.

    Defining an User:
    1. Click Create button.
    2. Enter login name of the user, which can have a maximum of 10 characters.
    3. Enter First name and last name.
    4. Other details such as location and group can be selected from the existing list, which is fetched from the database.
    5. Class: This is a number that defines the user’s authority level. The class number entered should be greater than zero.
    6. Supervisor: Users who are selected as supervisor gets listed in the dropdown box. The supervisors listed should be having a class, higher than the class of the user being defined. For example, if the user class is 4, then it should list all users whose class are less than 4 and who are defined as supervisor.
    7. 'Are you a Supervisor?': Select 'yes' in case the current user has to be defined as a supervisor and 'no' if the user is not a supervisor. Only when the user is defined as a supervisor, he/she can become a supervisor to other users. 'No' option is selected by default.
    8. Check the relevant 'availability status' of the user from the drop down list provided under availability. 'Available' option is selected by default.
    9. Enter password and confirm the password entered.

Click Apply button to save new or modified data of users.
Click Cancel button to cancel any modifications recently made.

To modify any user, double click on the specific user's login name from the list of existing users, displayed in the right side of the ‘Setup - Users’ screen. Click Edit to make the modifications.

To delete any user, double click on the specific user's login name from the list of existing users. Delete button is enabled on click of Edit button. On click of Delete button, confirmation is obtained and on acceptance of the same, the record is deleted.

Click Help button to view on line help page for ‘Setup - Users’.
Click “Close” button to close the present screen and return to the navigation screen of LAPS.
