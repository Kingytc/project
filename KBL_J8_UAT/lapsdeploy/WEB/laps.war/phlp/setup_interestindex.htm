Interest Index

To define an Interest Index:
This menu provides options to define interest rates that are to be used for various loan products. In addition to the interest rate, a label should also be provided. Up to 2 decimal places are allowed for the interest rate.
Interest description - The name of the interest rate has to be entered in this alphanumeric field. For instance, BoBTPLR, Absolute interest rate etc
Interest rate (%) - The interest rate for each of the names listed in the above field need to be entered in this numeric field.

Click Edit to make any modifications.
Click Apply button to save the new or modified interest rates.
Click Cancel button to cancel the operations made recently.
Click Help button to view online help page for ‘Setup - Interest Index’.
Click “Close” button to close the present screen and return to the navigation screen.
