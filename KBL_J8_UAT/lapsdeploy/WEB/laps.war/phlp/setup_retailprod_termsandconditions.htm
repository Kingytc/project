Terms & Conditions

    This is a centralized repository of terms and conditions, where you can create a set of terms and conditions for every loan category.
    Creating a new set of terms and conditions and attaching it to a loan category
    Select existing terms - Click this dropdown box
    Select “New” option from the dropdown list
    Term description - Enter the loan category/ sub-category for which you are creating a set of terms and conditions
    Enter one term/ condition at a time in the empty text box provided in the screen.
    Click “Add” every time you enter a new term/ condition
    Click “Cancel” to come out of the “new” mode. You will loose all entries made if you have not clicked “Add” before pressing Cancel
    Click “Close” to come back to the “Products” screen

    Editing an existing term/ condition
    You can see all the sets of terms and conditions in the dropdown box. Select existing terms. A number will represent each set.
    Select the set you wish to see
    The loan category and the list of terms/ conditions is displayed
    Click “Edit”
    Every item listed has a round radio button next to it. Click the button adjacent to the term/ condition you wish to change. It gets displayed in the text box
    Make necessary changes
    Click “Apply” to save changes
    Click “Delete” if you wish to delete the term/ condition
    Click “Close” to return to the Products screen
