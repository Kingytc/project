Other Rules

% of collateral Tab
Business rule ID - The loan product that was selected in “Step 5a” remains automatically selected. You can change it if required by just clicking the dropdown menu.
You can define upto five different ranges within which the percentage of collateral offered by the applicant might fall. Enter points for each range and description if needed. In our example, the housing loan:
Range from Range to Points Description
0 59.99 4
60 79.99 2
80 100 0
101 120 0
121 130 0

Enter description if required
Click “Save” to save changes
Click “ cancel” to cancel the operation. Data entered so far will not be saved in this case.
Click “close” to return to the Corporate Rating screen

You can define upto five different ranges of vehicle type based on their prices (luxury cars, mid segment, small cars, SUV’s etc) that the applicant is proposing to buy. Enter points for each range. In our example, the housing loan:
