Credit Rating Raster

Rating Tab
The system control will be in the sub-tab “Rating” by default
CMA - The CMA for which the credit Rating model is being created is chosen here
Click the dropdown box
The list of all CMA’s created is displayed
Select the CMA for which you are about to create a rating model
Select Rating - This is the alphanumeric field where you will create the Rating model title. From this point onwards, all relevant modules, parameters, scores are all attached to the model name created in this field
Enter the title of the rating model. For example, “Comprehensive Rating model”
Loan amount from & Loan amount to - Rating models differ according to the loan amount. The Rating model created in the above field is now attached to the loan amount through these two fields, namely, “Loan amount from and “Loan amount to”. While “loan amount from” is the beginning of the range, “Loan amount to” is the end of the range.
Industry Type - Select the industry type from the list box ie Trading or Manufacturing.
Click “Apply” to save the details entered
Click “Cancel” to cancel operations and also to change the existing mode. For example, the system is in “new mode by default in this screen. To be in “edit” mode, click “cancel”
Then click “edit” mode. In either case data entered or changes made will be lost if you click cancel
Click “Close” to go back to the “Commercial rating” screen
