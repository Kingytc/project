Powers
Click “edit”
BoB’s “User class” based sanctioning limit for loans is defined in this screen called “Powers”. For instance a BoB employee with User class 1 might have the authority to sanction loans up to Rs 1 Crore. The “user class” and “Limit” are defined as part of the product setup and hence the product will have to be renewed to make any changes.

    User designation - Enter the user group scale in this alphanumeric field. For e.g. Scale I Branch manager

    Powers - Enter the sanction power for a user belonging to the above mentioned scale. For e.g. Rs 1000000. You can enter up to 8 such records
    Click “Apply” to save the data entered
    You can click “Cancel” to Cancel the change carried out
    Click ““Close”” to return to the setup screen
