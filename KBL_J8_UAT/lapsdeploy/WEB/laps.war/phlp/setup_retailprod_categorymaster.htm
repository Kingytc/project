Category Master

    Creating a category
    By default the control will be in the “category” radio button
    Click “New”
    Now the cursor gets activated in the text box marked as “category”
    Type the name of the category. E.g. Car loan
    Click “Apply “
    Now the new entry is saved and hence the new category created. Multiple sub-categories can be created under each category

    Creating a sub-category
    Click the “sub-category” radio button
    A new dropdown box, Category appears
    Click the “New” button
    Click the dropdown box “category”
    Select the Category under which you wish to create sub-categories. For e.g. Car loan
    Now type the name of the new “sub-category” in the empty text box marked as sub-category. E.g. Takeover of car loan account
    Click “Apply”
    Now the entry is saved and hence a sub-category has been created under the selected category
    To create another sub-category, repeat the above steps
    You can click “Cancel” to cancel operations. Remember that you will lose all unsaved data
    Click “Close” to return to the setup screen

    To edit an existing category
    By default the control will be in the “category” radio button
    Click “Edit”
    Now the cursor gets activated in the text box marked as “category”
    Click the “?” adjacent to the category field. A list of all existing loan categories is displayed in a small popup window. Select the category, you wish to Edit e.g. Car loan
    The selected category appears in the text box
    Make necessary changes
    Click “Apply” to save changes
    You can click “Cancel” to Cancel the change carried out
    Click “Close” to return to the setup screen

    To edit an existing sub-category
    Click the “sub-category” radio button
    A new dropdown box, Category appears
    Click the “Edit” button
    Click the dropdown box “category”
    Select the Category under which the sub-category you wish to Edit is present. For e.g. Baroda Kanya Vivah Loan
    The selected sub-category name appears in the text box marked as sub-category.
    Make necessary changes
    Click “Apply” to save changes
    You can click “Cancel” to Cancel the change carried out
    Click “Close” to return to the setup screen
