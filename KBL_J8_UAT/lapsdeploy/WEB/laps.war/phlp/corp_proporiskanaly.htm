Risk Analysis
The “RISK ANALYSIS” tab is the most important in the proposal as the rating model gets executed here. While the required financial calculations, ratios are populated form the application form; the user in the “Risk analysis” screen only does the actual rating of the parameters. The following sub-tabs are found in the risk analysis tab:

Ratios
Cash Flow
Stress cash Flow
Funds Flow
Assessment
Rating Analysis
Rating model
Financial comments
Ratios
Ratios are calculated using the financial data entered in the corporate application form and the formulas defined in the “Formula definition” screen in the Setup module. The calculated ratios are displayed in this screen. The data displayed is non-editable. Click “Print” to print the screen
