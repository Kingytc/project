Credit Rating Raster

Section Tab
CMA - The CMA for which the credit Rating model is being created is chosen
Click the dropdown box
The list of all CMA’s created is displayed
Select the CMA for which you defined the “rating” & “module” in the previous sub-tabs
Select Rating - This is the dropdown where the Rating model is chosen. All relevant modules, sections, parameters, scores are all attached to the model name created in this field
Select module - This is the dropdown where the module is chosen. From this point onwards, all relevant sections, parameters, scores are all attached to this module name created in this field
Click the dropdown field and select the rating model, our example, “The comprehensive rating model”
Select section - This is the alphanumeric field where sections are created under each module is chosen. From this point onwards, all relevant parameters, sub-parameters and scores are all attached to this section name created in this field.
Enter the title for the section


-Do you want to distinguish weights - There is a drop down box in this field, where you can choose a “Yes” or “No”.
Choose “Yes” if you want to enter two sets of weightage details for “Working capital (WC)” and “Term loan (TL)”.
You will have to enter data in the following fields:
Working capital weight
WC maximum compulsory parameters,
WC maximum relevant parameters,
Term loan weight,
TL maximum compulsory parameters,
TL maximum relevant parameters
If you select “No” then the system asks for the maximum weightage, maximum compulsory parameters and maximum compulsory relevant parameters
Compulsory & relevant parameters - It is mandatory to create the exact number of parameters as defined in the above mentioned field. For instance, if the maximum compulsory parameters = 5 and maximum relevant parameters = 4 then 5 compulsory


Parameters and 4 relevant parameters have to be defined. At the borrower level the number of parameters should be same as the number defined here both of them are mandatory. The system maintains a control of the exact number and gives an alert when the “Rating analysis” is carried out.
Maximum weightage - The maximum weightage that the section created will carry in the Rating model is defined here. This score would be a composite weightage or the entire section, in other words an addition of all the parameters/ sub -parameter defined under the section
Compulsory/ Relevant parameters - The parameters that are likely to be defined under this section can be compulsory (parameters identified as mandatory for the particular Rating model/ industry) and relevant (these can be applicant specific)
Click “Apply” to save the details entered
Click “Cancel” to cancel operations and also to change the existing mode. For example, the system is in “new mode by default in this screen. To be in “edit” mode, click “cancel”
Then click “edit” mode. In either case data entered or changes made will be ;ost if you click cancel
Click “Close” to go back to the “Commercial rating” screen
