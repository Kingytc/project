Personal properties Tab

Click " Personal properties" sub-tab.
Enter data in the columns provided

Description: Enter a brief Description about the property owned for e.g. a typical description can be " Single bedroom apartment 900 Sq. ft in Andheri" or " 2001 model Santro Low mileage"
Survey No
City
Purchase Price: Enter the Purchase price of the property in Rupees
Market Value: Enter the current market value of the property, In other words, the value which the buyers would quote for the property as on date.
Valuation Date: Enter the date when the current market value of the property was ascertained.
Valuation done by: Enter the name of the Individual /agency name who ascertained the current market value of the property.
Applicant type - Click the drop down box
The following options are displayed: Applicant, Co-applicant, and Guarantor. If the co-applicant or guarantors are not present for this application then they do not figure on the list
Select the person to whom the personal property belongs
Already charged
Asset type - Click the drop down box
The following options are displayed: movable, immovable
Select the nature of the personal property
Collateral - Select the radio button next to the account details to offer it as collateral.
Once information is filled up, Click "Apply" to save the details entered
