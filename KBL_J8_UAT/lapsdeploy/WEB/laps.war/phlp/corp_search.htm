Creating a new corporate loan application
Click “New” in the corporate homepage
A search page is displayed
You will see two options, Corporate & SSI. There are 2 radio buttons adjacent to each of them. Select the radio buttons next to “Corporate” as we are going to create a corporate loan application
Now you will have to select the location entered in the “customer profile” for this particular applicant
Click the “?” next to each of the following: Zone, Region, Branch and select the location
Click “Search”
The relevant customer profile will be displayed
Click on the “Company Name”
A new corporate loan application form is displayed
Or
Enter the “Borrower Name” in the empty text box and click “Search”
”Reset” if you want to cancel the current search parameters and select totally new criteria
The relevant customer profile will be displayed
Click on the “Company Name”
LAPS will now ask if the applicant has applied for a corporate loan before
Click the radio button next to “Yes” if the applicant has already applied for a corporate loan with Bank of Baroda irrespective of the location where it was applied
Now the “Search by” box gets activated
Enter the “Application No.”
You can retrieve the old application along with the following details:
Applicant details - Click the box next to it
Principal details - Click the box next to it
Collateral details - Click the box next to it
Financial details - Click the box next to it
You can choose to retrieve all the above details or just any one or two. These details can be edited later if required
Click “Ok” to see the new loan application populated with chosen details
Or
If the applicant has never applied for a corporate loan before in Bank of Baroda, then select the radio button “No”
The new corporate loan application is displayed
