Documents Tab

Documents that are required to be submitted for this loan product is defined under “documents” in the Setup screen and this list gets displayed automatically when a new loan application is created using this product.
Documents can be submitted by the applicant and those submitted by the bank (security documents). So you will find 2 columns where you can select a list for both categories.

Click the “existing” button near “document category - applicant”
You will see a small popup window that will display the complete list of applicant documents (this has been already created in the “documents” under setup)
Select the documents that have to attached to this loan product
Click “Apply” to save the list
In the same way “document category - internal” which is the security documents list can also be attached
Click “existing” button
A popup window displays the complete list of all security documents
Select upto 20 documents that need to be attached for this loan product
Click “Apply” to save the list
You can click “Cancel” to cancel any changes carried out
Click “Close” to return to the setup screen
