Defaulters list

The Defaulters list published by the RBI has been fed into the system. Now every corporate client walking into the bank can be checked against this list using the search facility provided. Search can be carried out by 1 or more of the following fields; Bank name, Branch name, State, party name, Directors, Registered office address, Outstanding amount, Suit filed.

The search is not case sensitive but if the whole other search criteria (for e.g., Bank Name) is entered then appropriate punctuations needs to be entered. Thus, it is ok if you type, “Indian bank” but you cannot enter, “Indianbank”.

Entering just the first one or two letters of the search content can carry out a search. For e.g., If you type “In” in the “Bank Name” field and click “search”, LAPS will display all bank’s with names beginning with “In”. There is no facility to look for words that end with a particular letter.

If no criteria are entered but “Search “ button is hit then the system displays all available records. This way LAPS reduces risk

You can search the RBI defaulters list using the following search criteria:
Bank Name - Enter the bank name in the text box
Branch name - Enter the branch name in the text box
State - Enter the state to which the branch belongs to
Registered address - Enter the registered address of the client
Party name - Enter the name of the defaulting company that you are looking for
Outstanding amount - Enter the amount that is outstanding by the party to the bank
Director name - You can enter the name of one of the director’s of the defaulting company
Suit filed - Select “Yes” or “No” from a drop down box
Enter details of the entire set of search criteria or enter data in any of the fields
You will find a box that lists all the search criteria, named “Sort by”. For LAPS to accept the search criteria you have given, the search title must appear on the right side of this “sort by” box
Click the search criteria for which you have entered information in the text field above mentioned
Click “Add” - now the search criteria gets displayed on the right side of the “Sort by” box
Repeat this process if you have entered information in more than one search criteria
Click “Add all” if you have entered information in all the search fields.
Make sure that all of the search criteria are displayed in the right side of the “Sort by” box
Now click “Search list”
A popup window with the list of records that match the search criteria will be displayed
If there are no matching records, then LAPS will display a message, “No records to display”


Click “remove all” if you do not want to search using any of the criteria that is displayed in the right side box.
If you want to remove one of the items from the box then click the search item and then click “remove” instead of “remove all”


Click the “party” to see more details on the defaulting party. Information provided by the RBI in their defaulter’s list is what is displayed here.
