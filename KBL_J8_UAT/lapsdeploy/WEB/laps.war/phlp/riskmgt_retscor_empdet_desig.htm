Employment Details

Designation Tab
Business rule ID - The loan product that was selected in “Step 2a” remains automatically selected. You can change it if required by just clicking the dropdown menu.
There are 4 different designations (Self-employed is also included) that are evaluated in this screen. The applicant will get points depending on which category he falls under. The parameters are not editable but the points are soft in nature. In our example, for the housing loan
Designation Points Description
Executive 4
Managerial/ self-employed 2
Junior/ clerical 0
Other 0

Enter description if necessary
