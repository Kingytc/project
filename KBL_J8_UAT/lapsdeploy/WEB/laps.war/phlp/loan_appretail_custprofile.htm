Customer profile Search

    Click "Customer profile"
    Two options are available
    Edit existing applicant profile
    Create a new customer profile

    Edit existing customer profile
    If you want to edit an existing customer profile then use the search screen that appears. Records can be searched using the following criteria:
    First Name
    Last Name
    Branch Alpha code
    PAN/ GIR No
    Alternatively, Click "Search" - All available records will be displayed
    Click on the Name of the Applicant to see his/ her profile

    Create new customer profile
    If you want to create a new applicant, then Click "New Applicant" button
    A screen with 3 sub-tabs is displayed; Demographics, Income & expenses, Employment details
