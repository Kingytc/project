Credit Rating Raster

Parameter Tab
CMA - The CMA for which the credit Rating model is being created is chosen in this dropdown box
Click the dropdown box
The list of all CMA’s created is displayed
Select the CMA for which you defined the “rating, module & section” in the previous sub-tabs
Rating - Click the “?” next to the field
The list of rating models is displayed
Select the “rating” for which you are about to create parameters
Module - Click the “?” next to the field
The list of rating models is displayed
Select the “module” under the rating model for which you are about to create parameters
Section - Click the “?” next to the field
The list of rating models is displayed
Select the “section” in the module under the rating model for which you are about to create parameters
Criteria - Enter the actual description of the parameter in this alphanumeric field. For instance, “Performance of the Economy in Terms of GDP Growth and the Criteria - Effect of the growth of GDP of the Economy on the Industry”
Parameter type - The parameter defined could be differentiated as “compulsory” and “relevant”. The maximum number of relevant & compulsory parameters has already been setup under the “sections” tab and hence the system will allow only so much.
Click the dropdown menu
There will be two options, namely, “C” & “R”
Select whether the parameter described is a compulsory (C) or Relevant (R )
Is Sub-parameter available - There are two options, “Yes” and “No”. Choose “Yes” to create sub-parameters
Creating Sub-parameters
Click the radio button next to “Yes”
The minute you click “Yes”, the following fields will appear:
Loan type - The sub-parameter that you are about to create can apply to both loan types, (working capital and Term loan), or choose either one of them in this dropdown menu
Click the radio button adjacent to Sub-parameter
Parameter result type - Parameter score = Arithmetic mean of sub-parameter scores, Arithmetic mean = sub-parameter score/N, where N = number of sub-parameters. In case “Legend “ is chosen, Arithmetic mean is still calculated but the system compares the mean to the Legend table, which carries a score for different ranges. This Legend value is taken as the parameter score, i.e., Parameter score = Legend value for the arithmetic mean of sub-parameters
Click the dropdown field
Arithmetic mean is selected by default.
Select between the two options, namely, “Arithmetic mean”, “Legend”
Click the radio button adjacent to the word “Sub-parameter
Sub-parameter - Enter the sub-parameter title
Criteria - Enter the actual description of the parameter in this alphanumeric field. For instance,
Formula - You can choose a formula that will be applicable to this parameter.
Click the “?” next to the empty text box
A small popup window with a list of all available pre-defined formulas
Select the applicable formula
For example Parameter - Return on Capital Employed,

Formula - Historical ROCE
Value type - You can choose “Descriptive” in this field if you wish to have scoring descriptions and corresponding values.
Click the dropdown field
Choose “Descriptive”
Two sets of fields appear:
Description - You can enter upto five descriptions
Value - Enter the value corresponding to each of the description
Click “Apply” to save the details entered
Click the next tab to continue the setup process or click “”Close” to return to the “Commercial rating screen”
