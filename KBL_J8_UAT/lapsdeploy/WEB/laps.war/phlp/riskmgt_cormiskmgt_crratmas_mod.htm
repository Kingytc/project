Credit Rating Raster

Module Tab
CMA - The CMA for which the credit Rating model is being created is chosen here
Click the dropdown box
The list of all CMA’s created is displayed
Select the CMA for which you defined the “rating” in the previous sub-tab
Select Rating - This is the dropdown where the Rating model is chosen. All relevant modules, sections, parameters, scores are all attached to this model name created in this field.
Click the dropdown field and select the rating model, our example, “The comprehensive rating model”
Select module - This is the alphanumeric field where modules are created under the Rating models. From this point onwards, all relevant sections, parameters; scores are all attached to the module name created in this field.
Enter a name for the module
Maximum weightage - The maximum weightage that the module created will carry in the Rating model is defined here.
Enter the composite score for the entire module in this numeric field. This score would be a composite weightage for the entire module, in other words an addition of all the parameters/ sub -parameter scores

-Module access - The field displays a list box, that has 2 options “To the Risk management department” & “ To the branch”. This field defines the user department for this particular module. Choose “Risk management” if you want to restrict access to this module to only the Risk management department
Click “Apply” to save the details entered
Click “Cancel” to cancel operations and also to change the existing mode. For example, the system is in “new mode by default in this screen. To be in “edit” mode, click “cancel”
Then click “edit” mode. In either case data entered or changes made will be ;ost if you click cancel
Click “Close” to go back to the “Commercial rating” screen
