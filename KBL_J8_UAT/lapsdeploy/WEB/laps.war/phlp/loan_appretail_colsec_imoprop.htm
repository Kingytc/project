Coll. securities

Click " Coll. securities" tab.
You will see a screen with 2 Tabs, namely, Immovable property and Vehicle. Under each of these tabs, there are here sub-tabs: applicant, Co-applicant and Guarantor. Enter elaborate details on the assets, immovable property or vehicle that has been offered as a collateral in the “Financials” screen. Make sure details are entered based on the applicant type. For instance, if a house offered as collateral belongs to the co-applicant then information should be filled under the Co-applicant sub-tab under the “Immovable property” tab

Immovable property Tab
Click on the applicant type, which is suitable for the present situation. The screen and fields are the same in all the 3 sub-tabs, namely applicant, Co-applicant, Guarantor. Enter details in the following fields in the sub-tab:
Applicant type
Contact Person -Enter the name of the contact person for the property that has been offered as collateral. This is an alphanumeric field
Address - Enter the complete mailing address of the asset that has been offered as collateral including the city, State and pin code. These are alphanumeric fields while
Pincode alone is a numeric field
Last Property tax paid upto - Enter the date till which the property tax has been paid for the asset
Last water tax paid upto - Enter the date till which the property tax has been paid for the asset
Year Built - Enter the year in which the property built
Immovable property extent - Enter the area spread of the property (in Sq.ft). This is a mandatory numeric field
Survey No - The survey number for the property allotted by the local governing body, this is a unique identification number
Purchase price - Enter the actual purchase price of the property when its present owner bought it. This is a numeric field
Mortgage balance if any - If this property has been mortgaged with some institution then the balance left in the mortgage balance needs to be entered in this numeric field.
Appraised value - Enter the current appraised value for the property in this numeric field
