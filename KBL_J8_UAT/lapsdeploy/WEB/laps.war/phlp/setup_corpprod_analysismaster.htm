Analysis Master
The purpose of this screen is to enable the addition of ratios in the creation of QIS formats for various CMA ‘s.
Click “New” to add a new parameter to the QIS
Select the CMA Index and Row description and then click “Edit” if you wish to modify any existing parameter

    CMA Index - Click the dropdown box
    Every available CMA format is displayed (remember that these are attached to various industries in turn).
    Select the CMA title for which you are going to attach ratios

    Ratio description - Enter the new description in the given space or click the “?” and select from the existing list of ratio descriptions if you are modifying any of the parameters

    Row For - Click the dropdown box
    Select “QIS”
    Formula for Row - Click the “?” and select form the list of ratios already created

    Click “Apply” to save changes
    Click “Cancel” if you do not wish to retain changes carried out/ additions
    Click “Close” to return to the “Corporate products” page
