Note to MCB - Financials
Just as you can tailor-make financials for a particular industry, you can also create exclusive formats that constitute the “Note to the MCB (Management Committee Board)”. You can create a new set for each CMA title.
The system will be in “New” mode by default
    Category - In this field, You can choose the part of a CMA Index that needs to be created in a particular format to enable submission to the board of Directors for perusal. The categories available are:
    Financial - Note to Dir
    Cash Flow - Note to Dir
    Funds Flow - Note to Dir
    Ratios - Note to Dir
    Assessment - MPBF1 - Note to Dir
    Proposal - Note to Dir
    Creating a new category
    Click the dropdown box
    Choose from the categories available. This list is not editable.
    CMA Index - This is the field where you can link the category selected above to a CMA.
    Click the dropdown box
    Select the CMA for which you are creating the “Note to MCB” format
    Title - You can create sub-tabs in the “Financials” category using this field. LAPS does not limit you in creating these titles by predefining them instead you can create any number of heads depending on the requirement of your financial model. For example, In the CMA - M, you may have Cost of sales, Current Assets, Current liabilities, Networth, contingent liabilities etc as heads the “Financials”
    There are two options to fill this field:
    Click the “?” adjacent to the filed and select a title
    OR
    Enter a new title in the empty field
    Title field is not activated for other categories, namely, Ratios, Cash flow, Funds flow, Stress cash flow, Assessment - MPBF1, MPBF 2, MPBF 3, MPBF 4, PBF 1, PBF 2, PBF 3 and Turnover method

    Row description - You can enter the parameter/ row description in this alphanumeric field. You can create parameters for the sub-tabs created in the “Title” field. For example, You can enter a description here “Guarantees on behalf of other companies” under “Contingent liabilities” in the “Financials” part of CMA Risk management ”.
    There are two options to fill this field:
    Click the “?” adjacent to the filed and select the “title description”
    OR
    Enter a new title in the empty field
    Click “Apply” to save changes
    Click “cancel” if you want to cancel the operation
    Row For - This is again a drop down box, which displays the following options: heading and Formula. Click the option that is suitable for the respective parameter. For example:
    Select “Formula” if you want to attach a formula to any of the parameters. For example, If the parameter is “Net sales”
    Select “Heading” if you are creating a heading (title) so that you have parameters under it. For example, other non-operating income/ expenses
    S.No - Parameters are arranged in sequence by default but this order is editable. Enter the row number in which you want the respective parameter to appear
    Formula - This field appears only if the “Row for” field option is “calculated”. This is attached only after formula creation
