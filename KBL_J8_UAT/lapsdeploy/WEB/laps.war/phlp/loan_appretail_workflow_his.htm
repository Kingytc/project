Workflow
Click the "Workflow" tab
You will see 2 sub-tabs
Workflow denotes the various stages that an application goes through till the loan has been sanctioned. This tab has following three sub tabs, which are explained below:

History
Action
Remarks

History

Control will be in “History” sub-tab by default
It provides a log of how the application has moved during the processing of various actions.

Following details regarding the workflow history of the applicant is displayed automatically based on the "action". Following are the fields displayed:

From Flow point: Flow point number of the sender.
Sent By: User id of the user, who has forwarded this application.
Sent On: Date on which it is sent.
To Flow Point: Flow point number of the recipient, to whom it is forwarded.
Sent to: User id of the person, to whom it is sent for processing.
