Analysis Range
For every ratio that forms part of a QIS statement, you can define a range the borrower’s ratio might fall during the current year/ compare with previous year and give the bank’s comment’s for the ranges. For instance, if the current ratio in the QIS of a “pharmaceutical Industry” for the “current year” falls within the range of 2-4 then the bank’s comment could be “Excellent performance”.
Select the Ratio ID and Type before hitting ”Edit” to make any entries

    Ratio ID - Click the dropdown menu and select form the list of available ratio formulations

    Type - Click the dropdown box to select between Current year/ compare with previous year

    List - This is to refresh the screen as of now

    Range from - Range to - You can enter upto 4 sets of ranges for the Comments
