Industry Master

Creating a new industry

    Industry master
    System will be in "New" mode by default. In other words new industries can be created. There are three fields to fill-in details in this screen.

    • Industry name - Type the name of the industry you would like to create. For examples, "Heavy engineering". This is an alphanumeric field
    • BSR code - Type the unique identification code that is allotted for the industry in the above field. This is also an alphanumeric field so the code can consists of both alphabets and numbers
    • Products - Enter the products that the above mentioned industry produces. For example, for the industry, "Heavy engineering" the product could be "Brake linings". This is an optional alphanumeric field
    Click "Apply" to save your work
    Click "Cancel" if you do not want to save. Always click "Cancel" when you want to change modes.
    Click "Cancel" to come out of the "New" mode

    To edit existing records:
    Click the "?" adjacent to the field "Industry name"
    Select the industry you wish to Edit
    Now click "Edit" button to make modifications in the fields
    Make necessary modifications
    Click "Apply" to save changes

    Click "Cancel" if you do not want to save changes
    Click "Close" to close the screen and return to the "Corporate products" master
