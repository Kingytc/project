Other Rules

Lifestyle Tab
Business rule ID - The loan product that was selected in “Step 5a” remains automatically selected. You can change it if required by just clicking the dropdown menu.
There are eight pre-defined criteria based on which the lifestyle of the applicant is decided. The criteria are non-editable but you can define the points depending on the loan product.

Lifestyle Points Description
Car 0.5
Scooter 0.5
Fridge 0.5
Telephone 0.5
Computer 0.5
Cellular phone 0.5
Washing machine 0.5
Television 0.5

Enter description if required
Click “Save” to save changes
Click “ cancel” to cancel the operation. Data entered so far will not be saved in this case.
Click “close” to return to the Corporate Rating screen
