Co-applicant/ guarantor

With this screen you can create any number of co-applicants and guarantors for a loan application. Once a customer profile is defined, he/ she can be accepted as a co-applicant or guarantor for other loans.
Click “Add” to include this person as a co-applicant/ guarantor as specified in the “select applicant type” field
Select applicant type - Click the dropdown box
You can select between two options, namely, Co-applicant, Guarantor
Select one of them
Name of the person - Click the “?” next to this text box
The entire list of customer profiles is displayed in a small popup window
Select the required name
Relationship with the applicant - Enter the relationship between the quarantor/co-applicant and the applicant.
Click “Apply” to save the entries made
Click “Cancel to cancel the operation
