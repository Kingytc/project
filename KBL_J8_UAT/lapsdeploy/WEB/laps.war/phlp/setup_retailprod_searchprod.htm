Products
Click “Products”
You will see a search page with 3 options, namely, new products, existing and old products.

    New Products
    Select the radio button adjacent to this option to trigger the creation of a new loan product, you will see the new loan creation screens.

    Existing Products
    A fully configured loan product is deemed as ”completed” which means loan applications can be received and processed for that loan product. Such “partial”/ “completed” loan products can be viewed by selecting the radio button “existing products”. Various search criteria based on the product completion status (partial, complete) are also available.
    Loan products that are already in use in loan applications may have to be appended/ modified due to market conditions. To face such situations, any completed loan product (even if in use by many loan applications) can be pulled out from under the “existing products” option and then renewed. Which means a renewed application hence can be modified/ appended and used for new loan applications. Renewal can be carried out multiple times.

    Old Products
    By selecting this radio button, one can view all loan products that have been renewed.
