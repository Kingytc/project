NPA Financials
This screen forms the backbone of the automation process of NPA for corporate lending products. The complete financials including the balance sheet, Income statement & P& L statement, can be dynamically defined according to the industry to which the corporate applicant belongs.

By default, the “NPA Financial ” screen is in “new” mode. Following are the steps to be followed in creating the financial analysis model and the parameters:

    CMA Index - Choose the CMA title for which you are creating the financial model. This dropdown field displays all available CMA titles (those created under the “Industry master”) already attached to various industries. For example, the user chooses, “CMA risk management”

    Ratio description - The user can enter the parameter/ row description in this alphanumeric field. You can create parameters for the sub-tabs created in the “Title” field. For example, You can enter a description here “Guarantees on behalf of other companies” under “Contingent liabilities” in the “Financials” part of CMA Risk management ”.
    There are two options to fill this field:
    Click the “?” adjacent to the filed and select the “title description”
    OR
    Enter a new title in the empty field
    Click “Apply” to save changes
    Click “cancel” if you want to cancel the operation

    Row For - This is again a drop down box, which displays the following options: Financial, QIS. Click the option that is suitable for the respective parameter. For example:

    Select “Financial” if you are creating a heading (title) so that you have parameters under it. For example, other non-operating income/ expenses

    Formula - This is attached only after formula creation.
