Organization
Details of various organizations within the maximum five hierarchical levels can be entered using this submenu.

Sample screen as shown below is displayed on click of ‘Add Child’ button in Setup -> Organization page.

Details’ regarding ‘Level one’ is directly entered in the database. Child records to Level I and other levels can be added with different names through this menu.
Details displayed in this screen are explained below:

Note: Fields marked with ‘red’ * in ‘Setup - Organization’ screen are mandatory.
Organization code: This code is auto generated and it is a 15 digits code.

To define an organization:
1) Select the organization, for which child record is to be created, from the existing organization levels displayed in the right side of the screen.
2) Click Add Child button. This button enables you to add a child record for the selected parent organization.
3) Following details should be entered for an organization:
    Name of the organization.
    Organization ID and Transit Number of that organization.
    Address of the organization.
    City, state and pin code of the organization.
    Phone and Fax Number of the organization.
    URL of the organization and the Organization Head.
Click Edit button to modify existing values in the selected organization. However, organization code and short code of a particular organization cannot be modified.
Click Apply button to save the data entered or modified recently.
Click Cancel button to cancel the operations made recently.
A specific organization other than corporate can be deleted by clicking Delete button. But the organization chosen for deletion should have no child record, application and users.
Click Help button to view help for ‘ Setup - Organization’.
On click of “Close” button, confirmation is obtained and on acceptance of the same, navigation screen is displayed.
