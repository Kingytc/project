Click “New Application”
A search page is displayed - find below the screen with sample data


You will have to first indicate whether the loan application is for a ”Corporate loan” or “SSI loan”
Click the radio button next to “Corporate” to indicate that you are creating a corporate loan application

Step 2
You will have to enter the location of the applicant to create a new corporate loan application form.
Click the “?” next to Zone
Select the BoB Zone to which the region and in turn the branch where the application is being filed belongs
Click the “?” next to Region
Select the BoB Region to which the branch where the application is being filed belongs
Click the “?” next to Branch
Select the branch where the loan application is being filed
Click “Search”
Alternatively you can enter the “Borrower name” in the text box provided and click “Search”
Note - It is sufficient even if you enter the zone & region to carry out the search
LAPS displays all customer profile created by various applicants in the location entered above
Click on the “Applicant name”
Another page appears to find out if you (the applicant) have applied for a corporate loan before?
There are two radio buttons “Yes” & “No”
If you click “Yes”
Then you will have to enter the application number of the loan taken earlier
You can choose to retain one or all of the following information from that application, namely:
Applicant details
Principal details
Financial details
Collateral details
Click “Ok”
A new corporate loan application form is displayed
Sample screen

Click “No” if the applicant has never applied for a corporate loan before in BoB
A new corporate loan application form is displayed
