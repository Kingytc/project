Authority
This tab is used to fix the authority to approve / reject this loan product, skip flow points and also to fix authority for defining the interest rate for the loan product

    Minimum class for interest rate: Enter the minimum class to which a user should belong, in order to modify interest rate.
    - For example, if the maximum class is specified as ‘150’, only the users whose class is “150” or more can modify the interest rate for this loan product

    Approval/ Rejection: Enter the class value of the user who can approve/ reject loan application for this loan product. Users with the class value mentioned or higher can perform this action.
    - For example, if the class value of 200 is mentioned in this field, then users with class value 200 and more can approve / reject application for this loan product.

    Skip Flow points: This authority facilitates you to bypass one or more steps while processing an application for this loan product. Enter the Class value of the user who can skip flow points, during the workflow process.
    •Person with higher-class value than the class value specified can also skip flow points.
    •For example, if the class value of 200 is specified, then users with class value 200 and more than 200 can skip flow points for this loan product.

    Click “Apply” to save changes
    You can click “Cancel” to cancel changes carried out
    Click “Close” to return to the setup screen
