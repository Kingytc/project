Legend Marking
Each parameter carries certain marks in the Rating model. In the case of this parameter having multiple sub-parameters under it, the arithmetic mean of the marks that these sub-parameters score is taken as the parameter score
In case “Legend “ is chosen, Arithmetic mean is still calculated but the system compares the mean to the Legend table, which carries a score for different ranges. This Legend value is taken as the parameter score, i.e.., Parameter score = Legend value for the arithmetic mean of sub-parameters
“Legend marking” is the screen where various ranges for each Legend point scored by the applicant is created. For instance, if the parameter value of “Barriers to entry and the Criteria” = 2 (arithmetic mean of sub-parameters under Barriers to entry and the Criteria ). 2 is checked against the Legend marking table to see which Legend range it falls under. The “Legend point” against this range will go into the Rating model as the score for the parameter “Barriers to entry and the Criteria

To create a new “legend”
Click the dropdown box “Select legend”
Select the option “New”
Description - Enter the description for the new legend
Legend from & Legend to - Enter the range of legends. You can enter upto 4 ranges
Legend point - Enter the “points” that a parameter will earn if it falls into a legend range
To “Edit” an existing Legend
Click the dropdown box, “Select legend”
Select the range you wish to edit
Click “Edit”
Make necessary changes
Click “Save” to save the changes made
Click “cancel” to cancel the changes made
Click “Close” to close the screen and go back to the “commercial rating screen”
