Documents
Documents that are required to be submitted for each facility under the 4 facility heads can be Pre-defined during the setup. This list gets displayed automatically when a new loan application is created for the respective Facility.
There are two tabs, one "Applicant" to add additional documents to be submitted by the Applicant. While by choosing "By bank" additional internal documents from the banks side can be added. Documents once defined, as part of Setup cannot be deleted manner.

    Click the dropdown box "Select facility head"
    Select the facility head from the 4 standard non-editable options. The options are, Term finance funded, Term finance non-funded, Working capital funded, working capital non-funded.
    Click the dropdown box "Facility"
    Select the necessary "Facility"
    Click "Edit"
    Click the "Existing" button near "Document category - applicant"
    You will see a small popup window that will display the complete list of applicant documents (this has been already created in the "Documents"under setup)
    Select the documents that have to be attached to this loan scheme
    Click "Apply" to save the list
    In the same way "Document category - internal" which is the security documents list can also be attached

    Click "Existing" button
    A popup window displays the complete list of all security documents
    Select upto 20 documents that need to be attached for this loan scheme
    Click "Apply" to save the list
    You can click "Cancel" to cancel the change carried out
    Click "Close"to return to the setup screen
