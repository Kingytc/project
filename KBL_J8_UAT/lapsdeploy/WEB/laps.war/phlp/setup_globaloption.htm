Global options

This module allows you to define organization up to five levels. It has the following tab which is explained below:

    Organization
    Only user(s) who has administration right can enter/ modify information in this module. Other users can access one or more sub menus in this module depending upon the access rights defined in 'Setup - Groups'.

    The name for each of the organization levels should be mentioned against the corresponding fields and you can modify it as per the requirements.

    The organization level dropdown list box enables to select the number of levels. By default, two levels - Corporate and Regions are created. The number of levels can be increased any time. However the same cannot be decreased.
    Click Edit button to make modifications in ‘Setup - Global options’.
    Click Apply button to save the data entered or modified.
    Click Cancel button to cancel the operations made recently.
    Click Help button to view online help for ‘Setup - Global Options’.
    Click “Close” button to close the present screen and return to navigation screen of LAPS.

    Undelete Applications:
    This option enables you to restore the application(s) that are deleted. On click of Undelete Applications button, 'Get all deleted forms' window as shown in Error! Reference source not found. is displayed.

  • Two form types - ‘Retail’ and ‘Commercial’ are displayed as radio buttons. Select the desired form type by clicking the respective radio button.

  • To view all deleted applications in the chosen form type, without any search criteria, click ‘Search’ button.
  • You can filter the deleted applications by selecting any one of the following criteria:
  • For 'Retail' forms: Application number or last name or PAN number.
  • For 'Commercial' forms: Application number or company name or ID.

  • After specifying any one of the above search criteria, click 'Search' button. If there are applications that satisfy the given criteria, with corresponding check box on the left side.
    Out of these, one or more applications can be selected by selecting the corresponding check box. Then you can click Erase / Undelete button depending upon the requirement.

    • Click Erase to delete the application(s) permanently.
    Click Undelete to retrieve the application(s) back to its original position.
    On click of “Close” button, 'Get all deleted forms' screen is closed and 'Global options' screen as shown in the sample Error! Reference source not found. is displayed.
