Product Tab

    Creating a new retail loan Product

    Product Code - Product code is generated automatically for the product on successful entry of details under the 'products' tab is retrieved and displayed in this field. Till then LAPS displays, “New” in the field. This code is unique and is used to keep track of the product.

    Product for - This is a dropdown box. Select the appropriate category under which the loan product falls. The list is non-editable and is used for system internal tracking purposes by LAPS. The options available are:
    Housing loan
    Automobile loan
    Personal loan
    Consumer durables loan
    Computer loans
    Other loans

    Type of loan - Select the most suitable option from this dropdown list. Options available are:
    Overdraft
    Demand loan
    Term loan
    This list is non-editable and is used for internal purposes by LAPS

    Category - The list of loan categories that you have created in the “Category master” gets displayed in this dropdown box. Select the category to which the loan product you are creating will belong.

    Sub category: The list of sub-categories that belong to the category selected gets populated in this dropdown box. Select the appropriate sub-category
    No. of Term (Months) - Enter the maximum number of months by which the loan amount can be repaid.
    Range From and Range To - Enter the loan amount range for the loan product. For the same category / sub-category, there may be different range of loan amounts, making them separate loan products.
    Description: Enter a detailed description of the product,

    Product effective date: Enter the date from which the loan product can be offered to customers. You can also select the date from the 'calendar', which is displayed when you click the 'calendar' icon adjacent to this field. Date should be entered in DD/MM/YYYY format.
    Stop the product from current users - Select “Yes” in this dropdown field to stop this loan product for current users. Whenever required, it can be made available by selecting 'no' in this field provided loan applications have not been accepted for this loan product. In which case, you will have to renew the loan product and offer it as a new product.
    Select terms and conditions - Select the set of terms and conditions that you have created for this loan sub-category. You will be able to see the complete list of terms & conditions by clicking this dropdown list.

    Status: Status of data entered for the product is displayed in this field and the user cannot edit it. If details regarding a product are entered only for a few tabs, then 'partial' is displayed in this field. On successful entry of data under all the tabs, 'completed' status is displayed in this field.
    The defined product is available for application at any organization level provided that organization has been selected while creating the product.

    Note: On successful entry of the above details for a product, following details regarding the selected loan product is retrieved and displayed in top for rest of the five tabs excepting 'products' tab:
    Click “Apply” to save changes
    You can click “Cancel” to cancel the change carried out
    Click “Close” to return to the setup screen
